## ğŸš€ Getting Started

### Prerequisites

- Node.js 18.0.0 or higher
- npm 8.0.0 or higher

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/yourusername/pulse-robot.git
   ```

2. Navigate to project directory
   ```sh
   cd pulse-robot
   ```

3. Install dependencies
   ```sh
   npm install
   ```

4. Start the development server
   ```sh
   npm run dev
   ```
